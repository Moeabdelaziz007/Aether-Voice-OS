## Escalation Paths

### If Heartbeat Overdue

**Auto-Escalation**:
- 15 min late: Run heartbeat immediately upon next interaction
- 30 min late: Apologize and explain why missed
- 60 min late: Propose systemic fix to prevent future misses

**Common Causes**:
- Got distracted by urgent task
- Context loss without recovery
- Heartbeat timer not set properly

**Prevention**:
- Set multiple reminders
- Link heartbeat to user activity (not just time)
- Make heartbeat visible in activity stream

### If Repeated Failures

**After 3 consecutive missed heartbeats**:
1. Acknowledge pattern
2. Diagnose root cause
3. Propose architectural fix
4. Implement prevention mechanism
5. Document learning in MEMORY.md

## Best Practices

### Effective Heartbeats

✅ **DO**:
- Run consistently every 15 minutes
- Be honest in self-assessment
- Surface uncomfortable truths
- Propose concrete improvements
- Track metrics over time

❌ **DON'T**:
- Skip heartbeats without good reason
- Sugarcoat problems
- Ignore low scores
- Propose vague improvements
- Forget to document learnings

### Quality Indicators

**Good Heartbeat**:
- Specific findings (not vague)
- Actionable proposals
- Metrics updated
- Learnings documented
- Prevention mechanisms proposed

**Poor Heartbeat**:
- Generic checkmarks
- No concrete discoveries
- Metrics unchanged
- No new learnings
- Same issues repeatedly
